# Cinc Workstation devShell Flake

A flake that will install version 25.9.1094 of Cinc Workstation into a devShell.

### Usage
Clone the repo. 
Run the following command to enter the devShell:
``` bash
nix develop
```
You can now use cinc workstation commands. 

Based on https://github.com/kristofvandam/flake-cinc-workstation
